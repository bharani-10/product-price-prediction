# 🌟 Product Price Prediction App

Product Price Predictor**Machine Learning web application** that predicts product prices based on input features — built with a clean UI using Streamlit.


## 🚀 Live Demo

🔗 **Try the app here:**
https://price-prediction-app-ml.streamlit.app/

---

## ✨ Features

* 🤖 Machine Learning-based price prediction
* ⚡ Fast and real-time predictions
* 🎯 High accuracy using advanced models
* 📊 Clean and interactive UI
* 🧠 Multiple models comparison
* 💡 User-friendly input system

---

## 🧠 Model Performance

| Model             | R² Score | MAE (₹) | RMSE (₹) |
| ----------------- | -------- | ------- | -------- |
| Linear Regression | 0.72     | 1820    | 2450     |
| Decision Tree     | 0.93     | 620     | 980      |
| ⭐ Random Forest   | **0.97** | **380** | **610**  |
| Gradient Boosting | 0.96     | 420     | 680      |

👉 **Best Model:** Random Forest 🌲

## 📦 Installation

```bash
git clone https://github.com/bharani-10/product-price-prediction.git
cd product-price-prediction
pip install -r requirements.txt
```

## ▶️ Run the App

```bash
streamlit run app.py
```
